# HEXLET_test

Copy this repository

```bash
git clone git@github.com:Sp4nch/HEXLET_test.git
```

Enter the project directory:
```bash
cd HEXLET_test
```

To build Docker: 

```bash
docker build -t docker-practice .
```

To start Docker with server:

```bash
docker run -t docker-practice -p "3001:3000"
```
